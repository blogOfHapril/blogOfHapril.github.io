Hexo+Butterfly+Github搭建个人博客网站

发表于2024-08-15|更新于2024-09-04|Config环境搭建与使用

1.hexo+butterfly的部署

​ 零基础搭建个人博客！手把手教你用Hexo+GitHub+Zeabur打造属于你的网站!_哔哩哔哩_bilibili

.1 下载node.js（cmd）

​ 安装.msi时，勾选Automatically install the necessary tools

.2 npm安装其它包

.3 修改安装路径和缓存路径

​ D:\ProgramData下提前创建好node_global、node_cache，并给与文件夹访问权限

.4 设置环境变量

​ 系统变量path路径添加D:\ProgramData\node_global；新建NODE_PATH变量，值为D:\ProgramData\node_global

.5 cnpm（只推荐下载库时使用）

.6 初步部署Hexo（VSC git bash）

​ D:\ProgramData下创建hexo_github文件夹并用vsc打开，终端使用git bash

​ 若安装依赖时进度缓慢，ctrl+c终止，输入

​ hexo初步部署完成，测试

.7 Butterfly（vsc git bash）

​ 接着初步部署Hexo的步骤，下载主题文件

​ warn: 若出现SSL certificate problem，要么关闭网络加速，要么关闭网络验证（推荐使用时都执行）

​ 安装渲染器

​ 打开hexo->_config.yml，修改theme值为butterfly

​ 在hexo文件夹下创建**_config.butterfly.yml，并将themes->butterfly->_config.yml的内容复制到_config.butterfly.yml里，以后只需调整_config.butterfly.yml**

.8 部署

​ github创建仓库，命名为github用户名.github.io

​ hexo->_config.yml，修改deploy值为：

​ 回到git bash，下载部署插件

​ 后续更新发布
