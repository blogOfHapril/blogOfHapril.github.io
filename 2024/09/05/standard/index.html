基本规范

发表于2024-09-05|更新于2024-09-06|Config环境搭建与使用

1.tool

.1 path

（1）win

  • gui： D:\ProgramFiles
  • shell： D:\ProgramData
  • daily： D:\april
  • standalone：‪ E:\Bandizip

（2）ubuntu

  • /opt/
  • /home/ha/bin

.2 theme

（1）vsc

settings->autoDetectColorScheme，勾选 Window: Auto Detect Color Scheme
设置 Workbench 的三个选项：light： Quiet Light ；dark： Dark Modern

（2）typora

light： Github ；dark： d42ker-github

2.writing

.1 md（typora、csdn…）

  • 一级标题 ctrl+1 ，二级标题 ctrl+2
  • 正文内容用 （order） 来分隔
  • 内容评级：~，！，^，*，?
  • note，warning，recommend，链接：加粗&斜体
  • 内容被进行额外处理的，前后需各隔一个空格，以避免渲染错误
