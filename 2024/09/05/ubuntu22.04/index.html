ubuntu22.04安装及使用

  • 磁盘分配：0.5-16-240-120->376.5g
  • 在安装nvidia驱动前务必将bios中的secure boot（安全启动）关掉

1.ubuntu22.04安装

Windows 和 Ubuntu 双系统的安装和卸载哔哩哔哩bilibili

2.基本设置

.1 基本命令

（1） ctrl+l （或 clear ）：清空终端； ctrl+d （或 su ha 或 exit ）：退出root； ctrl+alt+t ：打开终端

（2）apt与apt-get（默认apt）

apt 结合了 apt-get 和 apt-cache 的功能，提供了更多通用命令，比如 apt install , apt remove , apt update , apt upgrade , apt search , apt show , apt autoremove 等。

apt-get 专注于软件包的安装和删除，而 apt-cache 用于查询和搜索软件包。

.2 添加清华源

ubuntu | 镜像站使用帮助 | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror

打开文件后把光标移到最开头，输入大写的I进入追加模式， ctrl+shift+v 将复制的镜像源内容粘贴，与文本原来内容隔两行以示区分，点击 esc ，并输入 :wq 保存并退出

.3 禁用nouveau并添加nvidia显卡驱动（不禁可能会导致黑屏）

NVIDIA显卡的Ubuntu驱动程序安装方法哔哩哔哩bilibili

（1）将nouveau加入黑名单

在打开的文件末尾隔两行添加 blacklist nouveau

（2）第一种方法（下载文件到本地再进行安装）， NVIDIA GeForce 驱动程序 - N 卡驱动 | NVIDIA ，操作系统：Linux 64-bit，下载类型：Production Branch

第一个弹窗选nividia，第二个选continue，第三个仍选continue，第四个允许安装32位，第五个允许优化，最后 reboot ，重启后输入 nvidia-smi 验证安装

（3）第二种方法（则使用官方PPA安装）

.4 添加中文输入法

Ubuntu22.04安装中文输入法——Google PinYin_ububtu22.04 googlepinyin-CSDN博客

Settings->region&language->Manage Installed Languages，Language Support窗口提示更新就install，确认Keyboard input method system为 fcitx 后，重启

终端输入 fcitx-configtool ，在fcitx输入法管理界面的Input Method点击加号添加 Google Pinyin ，在global config设置Trigger input Method位 L shift ，Candidate Word Number设置为9

.5 外观

（1）Ubuntu

Settings_Appearance，设置Style；Dock取消 Panel mode ，位置为Bottom，configure dock behavior两个都取消勾选

（2）Firefox

Themes：Alpenglow；插件：darkreader，infinity，saladict，tampermonkey

.6 系统时间同步，设置默认启动windows

双系统的启动菜单是ubuntu的grub引导器，从上到下由0开始编号，确定默认启动项；输入 sudo gedit /etc/default/grub ，修改GRUB DEFAULT的值，再输入 sudo update-grub 进行更新

.7 基本工具

todesk, qq,

3.软件安装

.1 VSC

Linux / Ubuntu上使用vscode编译运行和调试C/C++ - 知乎 (zhihu.com)

.2 Steam++（不建议）

steam++往hosts添加东西后反而会适得其反，而且安装过程十分繁琐

***【加速】Ubuntu 22.04 LTS Steam++ Watt Toolkit 加速 github_ubuntu steam++-CSDN博客***。

（1）下载linux_x64.tgz文件，打开终端

（2）点击加速，必须进行以下操作（若执行完后仍打不开，等一段时间即可）

  • 安装依赖

  • Host权限 - Watt Toolkit (steampp.net) ，Linux Host文件权限

  • Liunx 证书导入教程 - Watt Toolkit (steampp.net) ，Liunx 证书导入教程，打开 设置 - 隐私与安全 - 安全 - 证书 - 查看证书 ，选择 证书颁发机构( Authorities ) ，在 Watt Toolkit 的设置 - 通用设置 - 存储空间 - 打开 AppData 目录中的 Plugins/Accelerator 目录下， SteamTools.Certificate.pem 文件 ( 火狐支持 cer 或者 pem 格式导入 )，勾选 信任由此证书颁发机构来标识网站

.3 v2raya

Debian / Ubuntu - v2rayA

（1） recommend ：手动安装 deb 包

v2raya-apt/pool/main at master · v2rayA/v2raya-apt (github.com) ，下载v2ray_5.16.1_amd64.deb，v2raya_2.2.5.8_amd64.deb

（2）v2rayA，启动！

.4 alist+云盘

一键脚本 | AList文档 (nn.ci)

（1） recommend ：使用chorme打开网盘来获取cookies

要挂载子文件夹就打开子文件夹，此时地址栏最后面的一串数字几位子文件夹id（0表示根文件夹）

（2）alist，启动！
