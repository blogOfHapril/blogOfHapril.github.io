IDEA简单设置与使用

发表于2024-09-05|更新于2024-09-05|Config环境搭建与使用

1.设置

.1 idea基本设置

​ 设置jdk
File->Project Structure，设置jdk

​ 插件
ptg，Alibaba Java Coding Guidelines，CodeGlance Pro，GenerateAllSetter，GsonFormatPlus，JavaDoc，jclasslib Bytecode Viewer，LeetCode Editor，Presentation Assistant，Rainbow Brackets，Statistic

​ 外观
File->settings->Appearance下，Use custom font->Size(14)

​ IDEA包分层显示
主页，打开Project->Options->Tree Appearance，取消勾选Flatten Packages和Compact Middle Packages

​ 终端设置为cmd
File->Settings->Tools->Terminal->Application Settings，Shell path(cmd)

​ ctrl+/注释时，注释符取消行首显示：
settings->Editor->CodeStyle->Java下Code Generation->Comment Code取消即可

​ 代码提示取消大小写匹配：
settings->Editor->General->Code Completion下取消Match case

​ 取消双击shift打开全局搜索：
Settings->Advanced Settings，搜索double，勾选Disable double modifier key shortcuts

.2 IDEA+Git

​ 设置git
File->Settings->Version Control->Git->Path to Git executable(D:\ProgramFiles\Git\bin\git.exe)，返回主页，菜单栏->VCS->Enable Version Control Integration，选Git

​ IDEA登录github
File->Settings->Version Control->GitHub登录账号

​ 从github下载目标仓库（使用IDEA，就不推荐git bash下载了）
主页菜单栏Git->clone，在Repository URL下，URL输入github上目标仓库的SSH地址，并设置好项目下载的地址

​ 本地已有项目上传至github
主页双击shift键，输入Create Git Repository…，选择要上传的项目；随后项目右键Git->add，此时项目文件变成绿色，此时文件只是处于暂存区，并没有真正进入到本地仓库（版本库）中；项目右键Git->Commit Directory，勾选要上传的文件，并输入提交信息，此时项目文件从暂存区真正进入本地仓库，项目文件变成白色）

​ 上传项目到GitHub中
主页菜单栏->Git->GitHub->Share Project on GitHub，输入仓库名，添加相关描述，Share by处添加github账户（3.2已添加，此步无需重复），点击Share，即上传，上传成功后IDEA右下角会给出提示；返回github查看

​ 提交修改文件到GitHub
新增文件（红色），右键->Git->add，将新增的文件加入暂存区，此时文件变绿色，修改文件（蓝色）；在项目处，右键->Git->Commit Directory，查看有变动的文件并输入Commit Message，点击Commit and Push…

.3 algrithms4环境配置

​ 添加algrithms4代码依赖
主页，在项目目录src文件夹下创建edu.princeton.cs.algs4的Package，将解压后的algs4.jar文件夹（目录为algs4\edu\princeton\cs\algs4，注：不是解压后的所有文件）下的所有文件放于src创建的包（即里层的algs4）下，将解压后的algs4-data.zip文件夹下的所有文件放于src目录下

.4 快捷键

​ 生成变量来接收方法：Ctrl+Alt+V

​ Generate constructor, getter or setter method, etc: Alt+Insert（注意功能键未开启则需要额外键入Fn）

​ 控制台的显示和隐藏：Alt+4

​ 行注释：Ctrl+/；块注释：Ctrl+Shift+/

​ 抛出异常：Alt+Shift+Enter

​ 搜索类：ctrl+n，查看当前类的所有方法和成员变量：ctrl+f12、alt+7

​ 转到声明：ctrl+b

​ 提取出方法：选中要提取出方法的代码，ctrl+alt+m即可

​ 竖着选中：alt+左键选中
